# easy-snippet

VSCodeのユーザースニペット追加を簡単に

コードを選択して右クリックメニュー"Add Snippet"からスニペット追加できます.

This extension helps to add User Snippets.
By selecting code, you can add them to User Snippets from right-click menu.
